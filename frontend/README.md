# Electricity Forecasting Frontend

Modern React frontend for the Electricity Forecasting application.

## Features

- 🔐 Authentication (Login/Signup)
- 📊 Dashboard with consumption statistics
- 🔮 Consumption predictions
- 💬 AI-powered chat assistant
- 📱 Responsive design
- 🎨 Beautiful UI with Tailwind CSS

## Installation

```bash
cd frontend
npm install
```

## Development

```bash
npm run dev
```

The app will be available at `http://localhost:3000`

## Build

```bash
npm run build
```

## Tech Stack

- React 18
- React Router
- Vite
- Tailwind CSS
- Axios
- Recharts
- Lucide React Icons
